### Mob Finder Script (Put in Autoexe)
```lua
getgenv().Flames_Hub_Mob_Finder = true; -- // true = on (enabled) | false = off (disabled) \\ --
getgenv().Mob_Finder = {"The Arbiter","The Red Mist","Space Curse","Jotaro Kujo","Toji","Adjuchas","Deku"}; -- // List of NPCs to search for \\ --
getgenv().LoadingTime = 1; -- // Time to wait until it loads (Increase to 1.25 if device is slow) \\ --

loadstring(game:HttpGet("https://raw.githubusercontent.com/FlamesW/SakuraStand/main/MobFinder"))();
```

### Account Farmer Script (Put in Autoexe)
```lua
getgenv().Flames_Hub_Account_Farmer = true; -- // true = on (enabled) | false = off (disabled) \\ --
getgenv().Main_Account = "FullUsername"; -- // Change here to the name of ur main account (It will be selected as the farmer).
getgenv().LoadingTime = 1; -- // Time to wait until it loads (Increase to 1.25 if device is slow) \\ --

loadstring(game:HttpGet("https://raw.githubusercontent.com/FlamesW/SakuraStand/main/AccountFarmer"))();
```
